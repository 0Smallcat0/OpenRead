# OpenRead

> **A transparent, privacy-focused, "Bring Your Own Key" browser extension for immersive web and PDF translation.**
>
> **透明、隱私優先、"Bring Your Own Key" (自帶金鑰) 的沉浸式網頁與 PDF 翻譯擴充功能。**

---

## 🇬🇧 English

### ✨ Key Features

*   **🌐 Web & PDF Translation**: Supports translation for both webpages and local PDF files.
*   **🔑 BYOK (Bring Your Own Key)**: Use your own OpenRouter API key and freely choose the models you trust.

### 🚀 Installation Guide

Currently, this extension is installed via "Load Unpacked" mode.

1.  **Download** or **Clone** this repository to your local machine.
2.  Open your browser and navigate to `chrome://extensions`.
3.  Toggle on **Developer Mode** in the top-right corner.
4.  Click **Load Unpacked** (top-left) and select the folder containing this extension.


### ⚙️ Configuration (Get Your API Key)

To use the AI features, you need an API Key from OpenRouter. This gives you access to top-tier models at cost price without platform markups.

1.  **Visit OpenRouter**: Go to [OpenRouter.ai](https://openrouter.ai).
2.  **Sign Up / Log In**: Create an account.
3.  **Create Key**: Go to the "Keys" section and generate a new API Key.
4.  **Configure Extension**:
    *   Click the **OpenRead** icon in your browser toolbar.
    *   Paste your **API Key**.
    *   Select a **Model**.
    *   *(Note: Translation quality depends on the model you select via OpenRouter)*


### 📖 Usage Guide

#### 🌐 Web Translation
1.  **Select text** on any webpage.
2.  Click the **Floating Translation Icon** that appears.
3.  Read the immersive translation right next to your text.

#### 📄 PDF Translation
1.  **Drag and drop** a PDF file directly into your browser window.
2.  The extension will load it using its custom viewer.
3.  Select text to translate just like on a webpage.

### ⚠️ Critical Troubleshooting: PDF Support

> [!CANNOT ACCESS LOCAL FILES?]
> **You MUST enable file access permissions for PDFs to work.**

If dragging a PDF results in a blank page or error:
1.  Go to `chrome://extensions`.
2.  Find **OpenRead**.
3.  Click **Details**.
4.  Scroll down and toggle **ON** the specific setting: **"Allow access to file URLs"**.

## 🇹🇼 繁體中文 (Traditional Chinese)

### ✨ 主要功能

*   **🌐 網頁與 PDF 翻譯**: 支援網頁與本地 PDF 文件的翻譯功能。
*   **🔑 BYOK (自帶金鑰)**: 透過 OpenRouter 使用你自己的 API 金鑰，自由選擇你信任的模型。

### 🚀 安裝指南

目前本擴充功能透過 "載入未封裝項目 (Load Unpacked)" 方式安裝。

1.  **下載** 或 **複製 (Clone)** 本專案到你的電腦。
2.  開啟瀏覽器並前往 `chrome://extensions`。
3.  開啟右上角的 **開發者模式 (Developer Mode)**。
4.  點擊左上角的 **載入未封裝項目 (Load Unpacked)** 並選擇本擴充功能的資料夾。


### ⚙️ 設定指南 (取得 API 金鑰)

為了使用 AI 翻譯功能，你需要一組 OpenRouter API Key。這讓你以成本價使用頂級模型，無需支付額外平台費用。

1.  **前往 OpenRouter**: 訪問 [OpenRouter.ai](https://openrouter.ai)。
2.  **註冊 / 登入**: 建立一個帳戶。
3.  **建立金鑰**: 前往 "Keys" 頁面並產生一組新的 API Key。
4.  **設定擴充功能**:
    *   點擊瀏覽器工具列中的 **OpenRead** 圖示。
    *   貼上你的 **API Key**。
    *   選擇一個 **模型**。
    *   *(註: 翻譯品質取決於你在 OpenRouter 上選取的模型)*


### 📖 使用指南

#### 🌐 網頁翻譯
1.  在任何網頁上 **選取文字**。
2.  點擊出現的 **懸浮翻譯圖示**。
3.  即時檢視沉浸式翻譯結果。

#### 📄 PDF 翻譯
1.  直接將 PDF 檔案 **拖曳 (Drag & Drop)** 到瀏覽器視窗中。
2.  擴充功能會自動使用專屬閱讀器開啟。
3.  像在網頁上一樣選取文字即可翻譯。

### ⚠️ 重要疑難排解: PDF 權限

> [!無法讀取本地檔案?]
> **你必須開啟 "允許存取檔案網址" 權限，PDF 功能才能正常運作。**

如果你拖曳 PDF 後看到空白頁面或錯誤：
1.  前往 `chrome://extensions`。
2.  找到 **OpenRead**。
3.  點擊 **詳細資訊 (Details)**。
4.  向下捲動，找到並 **開啟** 此選項: **"允許存取檔案網址 (Allow access to file URLs)"**。
